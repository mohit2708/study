### Ques. Copy Dictionaries?
```python
thisdict =	{
"brand": "Ford",
"model": "Mustang",
"year": 1964
}

# using copy method:- 
mydict = thisdict.copy()
print(mydict)   # Output:-  {'brand': 'Ford', 'model': 'Mustang', 'year': 1964}

# using dict() method:- 
mydict = dict(thisdict)
print(mydict)   # Output:-  {'brand': 'Ford', 'model': 'Mustang', 'year': 1964}
```
