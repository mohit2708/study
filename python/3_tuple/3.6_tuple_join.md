### Join Tuples
* Using **plus(+)** operator
```python
# Join Two Tuples
tuple1 = ("a", "b" , "c")
tuple2 = (1, 2, 3)

tuple3 = tuple1 + tuple2
print(tuple3)

Output:- 
('a', 'b', 'c', 1, 2, 3)
```
* Using **Multiply(*)** operator
```python
fruits = ("apple", "banana", "cherry")
mytuple = fruits * 2

print(mytuple)
Output:- ('apple', 'banana', 'cherry', 'apple', 'banana', 'cherry')
```
