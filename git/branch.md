### Branch
* Show current branch name
```git
git branch

Output:
* master
```
* Show all branch
```git
git branch -r

Output:-
origin/HEAD -> origin/master
origin/MyLaravelProject
origin/code
origin/copy
```
```git
git branch -a

Output:-
* master
remotes/origin/HEAD -> origin/master
remotes/origin/MyLaravelProject
remotes/origin/code
remotes/origin/copy
```
