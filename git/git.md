Git:-
* git is a **version control system.** It was created by **Linus Torvalds** in 2005, and has been **maintained** by **Junio Hamano** since then.
* git helps you keep track of code changes.
* git is used to collaborate on code.
* Who made changes.
* Syntx:
```git
git --version
```
